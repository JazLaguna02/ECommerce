# E-Commerce Backend

A Node.js and MongoDB backend project for an e-commerce application.  
This project includes user authentication, password hashing with bcrypt, and product data models using Mongoose.

## Features
- User model with hashed passwords
- Password comparison for login
- Product model with pricing, inventory, category, and image fields
- MongoDB integration with Mongoose schemas

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- bcrypt

## Project Structure
- `User.js` – user schema and password hashing
- `Product.js` – product schema
- `app.js` – login/auth testing logic

## What I Learned
This project helped me practice backend data modeling, authentication basics, and working with MongoDB in a Node.js environment.
